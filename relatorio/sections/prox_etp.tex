\section{Conclusão}
\frame{
  \frametitle{Conclusão}
  \begin{block}{}
    Os ACO, SA, lógica \textit{fuzzy} e o AG não são apropriadas para resolver
    diretamente o problema. O ACO e SA podem ser empregados utilizando um
    classificador modelado com um CRF, onde atuariam na minimização do vetor de
    pesos associado a essa estrutura. A modelagem da solução utilizando lógica
    \textit{fuzzy} se mostrou muito complexa para este trabalho.
  \end{block}

  \begin{block}{}
    O futuro desse trabalho pode ser desenvolver uma inteligencia artificial
    baseada no \textit{MiniMax} que use a heurística pesquisada para simular o
    comportamento do adversário.
  \end{block}
}
