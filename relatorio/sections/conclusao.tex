\section{Conclusão}
\frame{
  \frametitle{Conclusão}
  \begin{block}{}
    \begin{itemize}
      \item Visa modelar o time adversário.
      \item Foram estudados: ACO, SA, GA, ANN, lógica fuzzy.
      \item Foi desenvolvido enunciado formal do problema. Importante: jogo não
        é completamente observável.
      \item Heurísticas analisadas criticamente: lógica fuzzy não é adequado
        devido à complexidade de introduzir a temporalidade, os métodos ACO, SA
        e GA servem para apoio de um algoritmo baseado em ANN\@.
    \end{itemize}
  \end{block}

  \begin{block}{}
    O futuro desse trabalho pode ser desenvolver uma inteligência artificial
    baseada no \textit{MiniMax} que use a heurística pesquisada para simular o
    comportamento do adversário.
  \end{block}
}
