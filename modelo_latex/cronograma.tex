\chapter{Cronograma}

\hspace{0.8cm}
Um plano de trabalho foi elaborado dividindo o projeto em 7 fases.
Segundo o cronograma original, foram estabelecidas as seguintes fases:

\begin{itemize}

\item A fase I, que consiste no estudo de conceitos teóricos sobre grafos e teoria espectral dos grafos.

\item A fase II, que consiste no estudo de ferramentas de banco de dados para armazenar os dados obtidos no PIBIT.

\item A fase III, que consiste na escolha da ferramenta de banco de dados a ser utilizada para armazenar os dados obtidos no PIBIT.

\item A fase IV, que consiste no estudo mais aprofundado da ferramenta escolhida.

\item A fase V, que consiste na criação do protótipo do banco de dados com as informações produzidas no PIBIT usando a ferramenta escolhida. 

\item A fase VI, que consiste em testes e criação da versão final do banco de dados.

\item A fase VII, que consiste emestudo de caso usando o banco de dados criado.
\end{itemize}

\hspace{0.8cm}
Até a presente data, a fase I foi concluída. Foram estudados conceitos básicos em Teoria de Grafos e Teoria Espectral de Grafos. A fase II está sendo desenvolvida paralelamente à fase III, pois ao mesmo tempo que um ferramenta de banco de dados é estudada, é feita uma comparação sobre as vantagens e desvantagens que ela apresenta sobre as outras. As demais fases dependem da conclusão das fases II e III para serem iniciadas;.
