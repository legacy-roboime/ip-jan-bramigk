\chapter{Conclusão}

\hspace{0.8cm}
Foram relatados neste texto os trabalhos desenvolvidos até a presente data. Embora tenhamos um grande volume de dados para analisar, nos propomos a desenvolver um banco de dados que permita a análise de forma simplificada destes dados, para pessoas que não tiveram contato tão a fundo com a obtenção destes dados.

\hspace{0.8cm}
Apesar de termos estudado duas ferramentas com características bem distintas, ain não foi possível definir qual será a melhor para se utilizar no trabalho, pois alguns fatores próprio da implementação devem ser analisados. É possível que venhamos a estudar ainda uma terceira ferramenta de banco de dados antes de decidir qual usar.
